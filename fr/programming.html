Dépot GitHub Documentation - > MDSV, pour Markdown Separated Values, est un langage de format de données qui ressemble à CSV, excepté que le caractère '|' est utilisé pour séparer les valeurs. Le but est de pouvoir analyser des tables situées dans un document Markdown.
Ceci a été fait pour me pratiquer à créer une librairie en Java et de la mettre dans le dépot central de Maven. La librarie ne supporte pas la sérialisation. Dépot GitHub - > Gitignore-modifier permet d'ajouter des patrons de fichiers à ignorer dans un fichier gitignore (pour Git). Le programme permet d'autres fonctions, mais l'objectif principal est de permettre de créer rapidement un bon fichier gitignore pour n'importe quel nouveau projet. Dépot GitHub ---
